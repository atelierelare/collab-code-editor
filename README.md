# Collab Code Editor

A real-time collaborative code editor with integrated chat functionality built using React, Node.js, Express, and Socket.IO. This project allows multiple users to join shared rooms, edit code simultaneously, and communicate in real time.

## Features

- Real-time collaborative code editing
- Room-based collaboration system
- Live synchronization of code across all users
- Integrated chat system for communication
- Create or join rooms using unique Room IDs
- Multiple users can work on the same code simultaneously
- Lightweight and responsive interface

## Tech Stack

Frontend:
- React
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Real-Time Communication:
- Socket.IO

## Installation

Clone the repository:
git clone https://github.com/your-username/collab-code-editor.git

Move into project directory:
cd collab-code-editor

Install backend dependencies:
cd server
npm install

Install frontend dependencies:
cd client
npm install

## Running the Project

Start backend server:
cd server
node index.js

Start frontend:
cd client
npm start

## Usage

1. Open the application in your browser
2. Create a new room or join an existing room using a Room ID
3. Share the Room ID with others
4. Start coding together in real time
5. Use the chat feature to communicate with collaborators
